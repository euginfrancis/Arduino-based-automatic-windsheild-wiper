# 🚗 Automatic Windshield Wiper

This Arduino project automates windshield wipers using a rain sensor. ☔️

## Features

- 🌧️ Rain detection
- ⚙️ Adjustable sensitivity

## Components

- Arduino Uno
- Rain sensor
- Servo motor
- Jumper wires
